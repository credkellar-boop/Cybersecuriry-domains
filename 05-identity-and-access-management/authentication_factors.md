# Authentication Factors

Strong authentication relies on combining multiple distinct factors (Multi-Factor Authentication or MFA).

*   **Type 1: Something you know** 
    *   Passwords, PINs, security questions.
*   **Type 2: Something you have** 
    *   Smart cards, hardware tokens, OTPs via SMS/Authenticator apps.
*   **Type 3: Something you are** 
    *   Biometrics (fingerprints, retina scans, facial recognition).
*   **Type 4: Somewhere you are** 
    *   Geolocation, IP address fencing.
*   **Type 5: Something you do** 
    *   Typing cadence, swipe patterns, signature dynamics.
