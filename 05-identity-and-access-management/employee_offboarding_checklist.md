# Secure Employee Offboarding Checklist

Timely revocation of access is a critical IAM control to prevent insider threats and unauthorized access by former employees.

## Immediate Actions (Upon Termination/Resignation)
*   [ ] Disable (do not immediately delete) the user's Active Directory/IdP account.
*   [ ] Terminate all active VPN and remote access sessions.
*   [ ] Reset passwords on any shared administrative or service accounts the user knew.
*   [ ] Revoke access to third-party SaaS applications (e.g., Salesforce, GitHub, AWS).

## Hardware & Physical Access
*   [ ] Collect all company-owned hardware (Laptops, phones, YubiKeys, hardware tokens).
*   [ ] Deactivate physical security badges and building access codes.
*   [ ] Wipe corporate data from personal devices (if BYOD/MDM is used).

## Data & Communication
*   [ ] Forward the user's email to their manager (if approved by HR/Legal).
*   [ ] Transfer ownership of critical files, Google Drive/OneDrive documents, and calendar events.
*   [ ] Remove the user from all internal distribution lists and communication channels (Slack, Teams).
