# Domain 5: Identity and Access Management (IAM)

## Overview
This domain centers on controlling who and what can access your systems and data. It covers the administration of access rights, authentication methods, and authorization frameworks.

## Key Topics
*   Physical and Logical Access Control
*   Identification and Authentication (MFA, Biometrics, SSO)
*   Authorization (RBAC, ABAC, MAC, DAC)
*   Identity as a Service (IDaaS) and Cloud IAM
*   Access Control Attacks
