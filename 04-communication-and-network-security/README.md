# Domain 4: Communication and Network Security

## Overview
This domain involves designing and protecting network architecture, securing communication channels, and maintaining the integrity of network components.

## Key Topics
*   Secure Network Architecture and Design (OSI and TCP/IP models)
*   Secure Network Components (Firewalls, Routers, Switches, IDS/IPS)
*   Secure Communication Channels (VPNs, IPsec, TLS, SSH)
*   Network Attacks and Countermeasures
