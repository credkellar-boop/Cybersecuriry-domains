# The OSI Model Cheat Sheet

Understanding the Open Systems Interconnection (OSI) model is critical for network security.

1.  **Physical:** Cables, hubs, bits (e.g., Ethernet, Fiber).
2.  **Data Link:** MAC addresses, switches, frames (e.g., ARP, VLANs).
3.  **Network:** IP addresses, routers, packets (e.g., IPv4, IPv6, ICMP).
4.  **Transport:** TCP/UDP, ports, segments (e.g., TCP, UDP).
5.  **Session:** Establishing and maintaining connections (e.g., NetBIOS, RPC).
6.  **Presentation:** Data formatting and encryption (e.g., JPEG, TLS).
7.  **Application:** Network applications and user interfaces (e.g., HTTP, FTP, DNS).
