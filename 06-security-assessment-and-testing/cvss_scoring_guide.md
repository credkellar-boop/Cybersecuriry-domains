# CVSS v3.1 Scoring Guide

The Common Vulnerability Scoring System (CVSS) provides a standardized way to capture the principal characteristics of a vulnerability and produce a numerical severity score.

## Severity Ratings
*   **None:** 0.0
*   **Low:** 0.1 - 3.9
*   **Medium:** 4.0 - 6.9
*   **High:** 7.0 - 8.9
*   **Critical:** 9.0 - 10.0

## Base Metric Group (The Exploit)
*   **Attack Vector (AV):** Network, Adjacent, Local, Physical. (How does the attacker reach it?)
*   **Attack Complexity (AC):** Low, High. (How difficult is it to exploit?)
*   **Privileges Required (PR):** None, Low, High. (Does the attacker need an account?)
*   **User Interaction (UI):** None, Required. (Does a victim need to click a link?)

## Base Metric Group (The Impact)
*   **Confidentiality (C):** None, Low, High.
*   **Integrity (I):** None, Low, High.
*   **Availability (A):** None, Low, High.
