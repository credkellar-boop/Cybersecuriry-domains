# Domain 3: Security Architecture and Engineering

## Overview
This domain focuses on the practice of designing, engineering, and building secure systems. It includes applying security models, utilizing cryptography, and understanding physical security principles.

## Key Topics
*   Secure Design Principles
*   Security Models (e.g., Bell-LaPadula, Biba)
*   Cryptography (Symmetric/Asymmetric encryption, Hashing, PKI)
*   Vulnerabilities in Security Architectures
*   Physical Security Design
