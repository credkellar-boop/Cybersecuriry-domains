# Standard Data Classification Matrix

Proper classification ensures data is handled with the appropriate level of security.

| Classification | Description | Example |
| :--- | :--- | :--- |
| **Public** | Disclosure will not cause harm to the organization. | Marketing materials, public website content. |
| **Internal** | Not for public release; minor impact if leaked. | Internal memos, organizational charts. |
| **Confidential** | Significant negative impact if disclosed. | Financial records, business plans, vendor contracts. |
| **Restricted** | Critical data; severe legal/financial impact if compromised. | PHI, PII, authentication credentials, source code. |
