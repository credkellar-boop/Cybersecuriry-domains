# Data Lifecycle Management
Effective security requires managing data through these phases:
* **Creation/Acquisition:** Data is generated or ingested.
* **Storage:** Data is placed in secure, authorized locations.
* **Usage:** Data is processed or accessed by authorized users.
* **Archival:** Data is moved to long-term, read-only storage.
* **Destruction:** Data is securely erased according to policy.
